Небольшая поисковая система, реализованная на C++.

Проект индексирует текстовые документы и выполняет полнотекстовый поиск с ранжированием результатов.

Система включает:

* индексатор документов
* поисковый сервер
* JSON API
* веб-интерфейс

---

# Возможности

* полнотекстовый поиск
* ранжирование результатов (BM25)
* параллельная индексация
* REST API
* веб-интерфейс
* сниппеты результатов
* просмотр документа
* подсветка найденных слов
* навигация по найденным совпадениям

---

# Архитектура

Проект состоит из нескольких модулей.

## InvertedIndex

Инвертированный индекс документов.

Функции:

* построение индекса
* хранение частоты терминов
* хранение длины документов
* быстрый доступ к спискам документов

---

## SearchServer

Поисковый сервер.

Функции:

* обработка поисковых запросов
* вычисление BM25
* ранжирование результатов
* нормализация score
* ограничение количества результатов

---

## JsonStorage

Модуль работы с JSON.

Функции:

* чтение `requests.json`
* сохранение результатов в `answers.json`
* сериализация результатов поиска

Используется библиотека **nlohmann::json**.

---

## TextUtils

Утилиты обработки текста.

Функции:

* токенизация
* нормализация текста
* обработка поисковых запросов

---

# Алгоритм ранжирования

Используется формула BM25.

score = idf * ((tf * (k + 1)) / (tf + k * (1 - b + b * dl / avgdl)))

где

tf — частота слова в документе
idf — обратная частота документа
dl — длина документа
avgdl — средняя длина документа

# Сборка проекта

Требования:

* C++17
* CMake
* компилятор GCC / Clang / MSVC

Сборка:

```
mkdir build
cd build
cmake ..
cmake --build .
```

# Запуск

! Для демонстрации в каталоге проекта имеется рабочий SearchEngine.exe

После сборки запустить сервер:

```
./SearchEngine
```

Открыть браузер:

```
http://localhost:8080
```
